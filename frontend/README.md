# ShiftWise

ShiftWise is a recruitment management platform built as a personal full-stack project to practice modern web development technologies.

The application allows recruiters to browse candidates and job offers through a clean dashboard interface while simulating common recruitment workflows.

---

## Features

### Dashboard
- Recruitment dashboard
- Statistics cards
- Total Users
- Candidates
- Recruiters
- Applications Sent
- Saved Jobs
- Top Candidates section

### Candidates
- View all candidates
- Candidate cards
- Skills display
- Years of experience
- Email information

### Recruiters
- View recruiter list

### Jobs
- Browse available jobs
- Search jobs by title, skill or location
- Filter jobs
  - All
  - Frontend
  - Backend
  - Remote
- Sort jobs
  - Title (A-Z)
  - Title (Z-A)
  - Experience
- View job details

### Job Details
- Complete job information
- Required skills
- Required experience
- Save a job
- Apply for a job
- Return to jobs list

### Applications
- View submitted applications
- Application date
- Remove applications

### Saved Jobs
- View saved jobs
- Remove saved jobs

### Data Persistence
- Applications remain saved after refreshing the browser
- Saved jobs remain saved after refreshing the browser

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install frontend dependencies

```bash
cd frontend
npm install
npm run dev
```

Install backend dependencies

```bash
cd backend
npm install
npm start
```
