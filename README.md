# ShiftWise

## Recruitment Management Platform

ShiftWise is a modern recruitment management platform developed with React and TypeScript.

The application allows recruiters to browse available jobs, search opportunities, apply for jobs, save vacancies for later and manage applications through an intuitive dashboard.

---

## Features

- Dashboard with statistics
- Candidate management
- Recruiter management
- Job listings
- Job details
- Search jobs
- Filter jobs
- Sort jobs
- Apply to jobs
- Save jobs
- LocalStorage persistence
- Responsive layout
- Modern UI

---
